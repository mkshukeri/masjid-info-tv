# Masjid Info TV

## Paparan Info Masjid & Surau Tanpa Perlu Server

**Masjid Info TV** ialah sistem paparan TV ringkas untuk masjid dan surau. Ia boleh digunakan untuk memaparkan info masjid, waktu solat, QR derma, pengumuman berjalan, hebahan program dan slaid dakwah melalui skrin TV.

Sistem ini direka supaya mudah digunakan oleh pihak masjid tanpa perlu pemasangan server, database, hosting atau sistem yang kompleks.

Cukup buka fail:

```text
index.html
```

menggunakan browser seperti Google Chrome, Microsoft Edge atau browser Android TV.

---

# 1. Apa Yang Ada Dalam Masjid Info TV

## 1.1 Paparan Waktu Solat

Masjid Info TV boleh memaparkan waktu solat harian seperti:

* Subuh
* Syuruk
* Zohor
* Asar
* Maghrib
* Isyak

Paparan waktu solat turut menyokong:

* Nama masjid
* Zon waktu solat
* Jam semasa
* Tarikh
* Waktu solat seterusnya
* Highlight waktu solat aktif / seterusnya
* Sambungan API waktu solat Malaysia

Data waktu solat boleh disambungkan kepada API e-Solat/JAKIM atau API lain yang ditetapkan di halaman konfigurasi.

---

## 1.2 QR Derma / QR Sedekah Masjid

Masjid boleh memaparkan QR derma atau QR sedekah pada skrin TV supaya jemaah mudah membuat sumbangan.

Ruang ini sesuai digunakan untuk:

* QR derma masjid
* QR tabung pembangunan
* QR sedekah Jumaat
* QR tabung kebajikan
* QR program khas
* Pautan derma digital

Pihak masjid boleh menukar gambar QR dan mesej derma melalui halaman konfigurasi.

---

## 1.3 Info & Dakwah

Ruang utama paparan boleh digunakan untuk menyampaikan kandungan dakwah dan maklumat masjid.

Contoh kandungan:

* Hadis harian
* Mutiara kata dakwah
* Ayat Al-Quran terpilih
* Hebahan kuliah
* Poster program masjid
* Jadual kelas pengajian
* Pengumuman aktiviti komuniti
* Gambar atau visual hebahan

Slaid boleh bertukar secara automatik mengikut tempoh masa yang ditetapkan.

---

## 1.4 Slaid Dakwah & Hebahan

Masjid Info TV menyediakan fungsi slaid untuk memaparkan beberapa kandungan secara bergilir.

Pihak masjid boleh:

* Tambah slaid baharu
* Masukkan tajuk slaid
* Masukkan teks dakwah / hadis / hebahan
* Masukkan sumber atau keterangan khas
* Pilih gambar latar
* Tetapkan tempoh tukar slaid
* Padam slaid lama

Fungsi ini sesuai untuk masjid yang ingin memaparkan kandungan harian tanpa perlu design poster setiap masa.

---

## 1.5 Pengumuman Berjalan

Masjid Info TV mempunyai ruang pengumuman berjalan di bahagian bawah skrin.

Contoh pengumuman:

```text
Kuliah Maghrib malam ini selepas solat Maghrib.
```

```text
Tabung pembangunan masjid dibuka bagi membiayai naik taraf dewan utama.
```

```text
Terima kasih atas segala sumbangan ikhlas anda. Semoga Allah melipatgandakan ganjaran.
```

Pengumuman ini sesuai untuk makluman ringkas yang ingin dilihat oleh jemaah secara berulang.

---

## 1.6 Halaman Konfigurasi

Masjid Info TV menyediakan halaman konfigurasi untuk mengubah tetapan tanpa perlu edit code.

Antara tetapan yang boleh diubah:

* Nama masjid
* Zon waktu solat
* API waktu solat
* Label QR derma
* Gambar QR derma
* Pautan derma
* Mesej derma
* Pengumuman berjalan
* Slaid dakwah
* Tempoh tukar slaid
* Offset minit waktu solat jika diperlukan

---

## 1.7 Export & Import Setup

Semua tetapan boleh dieksport sebagai fail backup.

Fungsi ini penting kerana sistem menyimpan data di dalam browser storage.

Dengan fungsi backup ini, pihak masjid boleh:

* Simpan salinan tetapan
* Pindahkan tetapan ke komputer lain
* Pulihkan tetapan jika browser data dibersihkan
* Simpan backup selepas membuat perubahan penting

---

## 1.8 Tema Warna Paparan

Masjid Info TV turut menyediakan pilihan **tema warna paparan** supaya paparan boleh disesuaikan mengikut suasana masjid, jenis TV dan tahap kecerahan skrin.

Tema boleh dipilih terus melalui halaman konfigurasi tanpa perlu mengubah kod sistem.

Pilihan tema yang tersedia termasuk:

### Preset Tema Lembut

* Hijau (Asal)
* Biru Gelap
* Maroon
* Ungu Gelap
* Kayu Teak
* Arang / Emas

### Tema Kontras Tinggi

Tema ini sesuai untuk TV yang cerah, ruang paparan yang luas, atau keadaan di mana teks perlu lebih jelas dari jarak jauh.

* Hijau Stark
* Biru Neon
* Maroon Stark
* Ungu Kontras

Fungsi ini memudahkan pihak masjid memilih tema yang lebih selesa dilihat oleh jemaah tanpa perlu membuat perubahan teknikal.

# 2. Kelebihan Masjid Info TV

## 2.1 Tidak Perlu Server

Sistem ini tidak memerlukan:

* Server
* Hosting
* Database
* WordPress
* Backend system
* NPM
* Node.js
* Installation teknikal

Cukup buka fail **index.html** di browser.

---

## 2.2 Mudah Digunakan Oleh Masjid

Pihak masjid tidak perlu tahu coding untuk mengubah kandungan asas.

Sebahagian besar perubahan boleh dibuat melalui halaman konfigurasi.

---

## 2.3 Sesuai Untuk TV Masjid / Surau

Sistem ini sesuai digunakan pada:

* Smart TV
* Android TV box
* Mini PC
* Laptop
* Komputer pejabat masjid
* Skrin paparan di ruang solat
* Skrin paparan di lobi masjid

---

## 2.4 Boleh Digunakan Secara Offline

Paparan asas boleh terus digunakan walaupun tiada internet, selagi data telah disimpan di browser.

Internet hanya diperlukan untuk mendapatkan data waktu solat terkini daripada API.

---

# 3. Cara Buka Sistem

1. Buka folder **Masjid Info TV**.
2. Cari fail **index.html**.
3. Klik dua kali fail tersebut.
4. Sistem akan dibuka di browser.
5. Untuk paparan TV, tekan **F11** atau gunakan fullscreen mode.
6. Biarkan browser terbuka untuk paparan berterusan.

---

# 4. Cara Buka Halaman Konfigurasi

1. Buka paparan utama Masjid Info TV.
2. Klik butang atau ikon konfigurasi jika disediakan.
3. Halaman konfigurasi akan dibuka.
4. Buat perubahan yang diperlukan.
5. Semak semula paparan utama selepas membuat perubahan.

---

# 5. Cara Ubah Nama Masjid

1. Pergi ke halaman konfigurasi.
2. Cari bahagian **Identiti & Derma Masjid**.
3. Masukkan nama masjid.
4. Semak nama masjid di paparan utama.
5. Selepas selesai, buat backup dengan klik **Export Setup**.

Contoh:

```text
Masjid Contoh Al-Falah
```

---

# 6. Cara Ubah Zon Waktu Solat

1. Pergi ke bahagian **Zon & API Solat Malaysia**.
2. Pilih zon JAKIM yang sesuai.
3. Sistem akan menggunakan zon tersebut untuk paparan waktu solat.
4. Klik butang muat turun data API terkini jika disediakan.
5. Semak waktu solat di paparan utama.

Contoh zon:

```text
WLY01 - Kuala Lumpur / Putrajaya
```

```text
MLK01 - Melaka
```

---

# 7. Cara Ubah QR Derma Masjid

1. Pergi ke bahagian **Identiti & Derma Masjid**.
2. Masukkan label QR derma.
3. Masukkan mesej ringkas QR.
4. Muat naik gambar QR derma.
5. Semak paparan QR di skrin utama.

Contoh label:

```text
QR Sedekah Masjid
```

Contoh mesej:

```text
Tabung Masjid & Sistem Audio Baru
```

Pastikan gambar QR jelas, tidak kabur dan mudah diimbas dari jarak jauh.

---

# 8. Cara Tambah Pengumuman Berjalan

1. Pergi ke bahagian **Pengumuman Berjalan Jemaah**.
2. Masukkan teks pengumuman.
3. Klik **Tambah**.
4. Pengumuman akan dipaparkan di footer paparan utama.

Contoh pengumuman:

```text
Kuliah Maghrib malam ini selepas solat Maghrib.
```

```text
Tabung pembangunan masjid dibuka bagi membiayai naik taraf dewan utama.
```

---

# 9. Cara Tambah Slaid Dakwah & Hebahan

1. Pergi ke bahagian **Slaid Dakwah & Hebahan**.
2. Tetapkan tempoh tukar slaid dalam saat.
3. Masukkan tajuk slaid.
4. Masukkan sumber atau keterangan khas jika ada.
5. Masukkan teks dakwah, hadis atau hebahan.
6. Pilih gambar latar jika diperlukan.
7. Klik **Daftar Slaid**.
8. Semak paparan utama.

Contoh tajuk slaid:

```text
Mutiara Hadith
```

Contoh teks:

```text
Sesungguhnya yang memakmurkan masjid-masjid Allah hanyalah orang-orang yang beriman kepada Allah dan hari akhirat.
```

---

# 10. Cara Simpan Perubahan

Setiap perubahan akan disimpan secara automatik di dalam **browser storage**.

Ini bermaksud tetapan disimpan di browser yang sama pada peranti yang sama.

Contoh:
Jika tetapan dibuat menggunakan Google Chrome di mini PC masjid, tetapan tersebut akan kekal di Google Chrome pada mini PC tersebut.

---

# 11. Perkara Penting Tentang Browser Storage

Browser storage bukan database kekal.

Tetapan boleh hilang sekiranya:

* Browser history/cache dibersihkan
* Browser ditukar
* Peranti ditukar
* Browser di-reset
* Sistem operasi diformat
* Mode incognito/private digunakan
* Storage browser dibersihkan oleh software cleaner

Oleh itu, setiap kali selepas membuat perubahan penting, pihak masjid sangat disarankan untuk membuat backup.

---

# 12. Cara Backup Tetapan

Selepas selesai membuat perubahan:

1. Pergi ke halaman konfigurasi.
2. Klik butang **Export Setup**.
3. Sistem akan memuat turun fail backup tetapan.
4. Simpan fail backup tersebut di tempat yang selamat.

Tempat simpanan yang disarankan:

* Pendrive
* Google Drive
* OneDrive
* Folder backup masjid
* Komputer pejabat masjid

Cadangan nama fail backup:

```text
backup-masjid-info-tv-2026-06-13.json
```

---

# 13. Cara Restore / Import Tetapan

Jika tetapan hilang atau ingin pindahkan ke peranti lain:

1. Buka halaman konfigurasi Masjid Info TV.
2. Klik butang **Import Setup**.
3. Pilih fail backup yang pernah dimuat turun.
4. Sistem akan memuat naik semula tetapan.
5. Semak paparan utama untuk pastikan maklumat telah dipulihkan.

---

# 14. Bila Perlu Buat Backup

Backup disarankan setiap kali selepas membuat perubahan seperti:

* Tukar nama masjid
* Tukar zon waktu solat
* Tukar QR derma
* Tambah pengumuman
* Padam pengumuman
* Tambah slaid dakwah
* Padam slaid dakwah
* Tukar API waktu solat
* Tukar tetapan masa paparan slaid

Amalan terbaik:

```text
Setiap kali selesai edit, terus klik Export Setup.
```

---

# 15. Cadangan Penggunaan Harian

Untuk penggunaan harian di masjid:

1. Hidupkan TV dan mini PC / laptop.
2. Buka fail **index.html**.
3. Pastikan paparan utama muncul.
4. Tekan fullscreen.
5. Semak waktu solat.
6. Semak QR derma.
7. Pastikan pengumuman berjalan dipaparkan dengan betul.

---

# 16. Cadangan Operasi Mingguan

Sekurang-kurangnya seminggu sekali:

1. Semak waktu solat.
2. Semak QR derma masih boleh diimbas.
3. Kemas kini pengumuman.
4. Tambah slaid program terkini.
5. Buang pengumuman lama.
6. Klik **Export Setup** untuk simpan backup terbaru.

---

# 17. Nota Penting

Masjid Info TV menyimpan tetapan di dalam browser. Oleh itu, jangan clear browser data tanpa membuat backup terlebih dahulu.

Jika ingin pindahkan sistem ke komputer lain, gunakan fungsi **Export Setup** dan **Import Setup**.

Untuk keselamatan, simpan sekurang-kurangnya satu salinan backup di luar komputer paparan utama.

---

* Masjid Info TV ialah sistem paparan info TV untuk masjid dan surau
